For instance, we want to create a blog in rails, use the command:
```
~$ rails new blog
```

# Start the server
```
~$ cd blog
~$ bin/rails server
```

Access it in http://127.0.0.1:3000. See details in [this link](https://guides.rubyonrails.org/getting_started.html#creating-the-blog-application)
